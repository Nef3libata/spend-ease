# SpendEase

A smart financial tracker for international residents in Germany. Helping you understand your spending, optimize grocery costs, and save for what matters.

## Why SpendEase?

Moving to Germany means navigating unfamiliar stores, prices, and financial systems. SpendEase eliminates the guesswork by tracking your expenses and helping you reach savings goals faster.

## Installation

```bash
git clone https://github.com/Nef3libata/spend-ease.git
cd spend-ease
uv pip install -e .
```

## Usage

```bash
uv run -m spend_ease
```
